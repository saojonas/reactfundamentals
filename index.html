Existem várias maneiras de estilizar elementos em React e CSS. Vamos explorar as diferenças entre CSS tradicional, módulos CSS e estilos inline.

CSS Tradicional

Estilos definidos em arquivos .css separados, aplicados globalmente.

Módulos CSS

Estilos com escopo local usando arquivos .module.css, evitando conflitos.

Estilos Inline

Estilos aplicados diretamente nos elementos via propriedade style.

Demonstração de Cores

Experimente mudar as cores abaixo:

Este é um exemplo de elemento estilizado. As cores serão alteradas quando você clicar em "Aplicar".

Código Exemplo

// CSS Tradicional (arquivo styles.css)
.container {
  background-color: #4f46e5;
  color: white;
  padding: 1.5rem;
  border-radius: 0.5rem;
}

// Módulo CSS (arquivo Styles.module.css)
.container {
  composes: base from global;
  background: linear-gradient(to right, #4f46e5, #c084fc);
}

// Estilo Inline (no componente)
function ColoredBox() {
  return (
    <div style={{
      backgroundColor: '#4f46e5',
      color: 'white',
      padding: '1.5rem',
      borderRadius: '0.5rem'
    }}>
      Conteúdo estilizado
    </div>
  );
}

Dica Prática

Para projetos maiores, prefira módulos CSS ou bibliotecas como TailwindCSS para manter os estilos organizados e evitar conflitos de classes.

Posicionamento com Flexbox

Flexbox é um modelo de layout unidimensional que permite distribuir espaço entre itens em uma interface e alinhá-los de maneira poderosa.

Controles Interativos

10px

Demonstração

1
2
3
4

Código Exemplo

/* Container Flexbox */
.container {
  display: flex;
  justify-content: flex-start; /* Alinhamento horizontal */
  align-items: flex-start;    /* Alinhamento vertical */
  flex-direction: row;        /* Direção dos itens */
  gap: 10px;                  /* Espaço entre itens */
}

/* Itens Flex */
.item {
  width: 64px;
  height: 64px;
  background-color: #4f46e5;
  border-radius: 0.5rem;
}

/* Responsividade */
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

Dica Prática

Use flex: 1 em itens para que eles preencham o espaço disponível proporcionalmente. Combine com flex-wrap: wrap para layouts responsivos que quebram em várias linhas.

Posicionamento Absoluto

O posicionamento absoluto remove um elemento do fluxo normal do documento e o posiciona em relação ao seu ancestral posicionado mais próximo.

Demonstração Interativa

50px
50px
Box

Este é o container relativo (position: relative). O quadrado vermelho está com position: absolute.

Código Exemplo

<!-- HTML -->
<div class="relative-container">
  <div class="absolute-box"></div>
  <p>Conteúdo normal</p>
</div>

/* CSS */
.relative-container {
  position: relative; /* Ancestral posicionado */
  height: 200px;
  background-color: #f3f4f6;
}

.absolute-box {
  position: absolute;
  top: 50px;  /* Distância do topo */
  left: 50px; /* Distância da esquerda */
  width: 80px;
  height: 80px;
  background-color: #ef4444;
}

Dica Prática

Evite usar position: absolute para layouts principais. Reserve para elementos como tooltips, modais ou elementos de sobreposição. Sempre defina um ancestral com position: relative para conter o elemento absoluto.

Imagens via URL

As imagens podem ser carregadas de URLs externas ou do seu projeto. A propriedade object-fit controla como a imagem se ajusta ao seu container.

Demonstração

Imagem de exemplo do Unsplash

Código Exemplo

// Componente de Imagem em React
function ResponsiveImage() {
  return (
    <div className="image-container">
      <img 
        src="https://exemplo.com/imagem.jpg" 
        alt="Descrição da imagem"
        className="image-style"
      />
    </div>
  );
}

/* CSS */
.image-container {
  width: 100%;
  height: 300px;
  background-color: #f3f4f6;
}

.image-style {
  width: 100%;
  height: 100%;
  object-fit: cover; /* ou contain, fill, etc. */
  border-radius: 0.5rem;
}

Dica Prática

Sempre inclua o atributo alt para acessibilidade. Para imagens responsivas, considere usar o elemento <picture> com múltiplas fontes ou a propriedade CSS srcset.

Bordas Arredondadas

A propriedade border-radius permite criar cantos arredondados em elementos. Você pode especificar valores individuais para cada canto.

Demonstração Interativa

8px

Experimente ajustar o border-radius e a cor!

Exemplos de Bordas

Código Exemplo

/* Bordas arredondadas básicas */
.rounded-sm {
  border-radius: 0.125rem; /* 2px */
}

.rounded-md {
  border-radius: 0.375rem; /* 6px */
}

.rounded-lg {
  border-radius: 0.5rem; /* 8px */
}

.rounded-full {
  border-radius: 9999px; /* Círculo perfeito */
}

/* Bordas individuais */
.custom-rounded {
  border-top-left-radius: 1rem;
  border-top-right-radius: 0.5rem;
  border-bottom-right-radius: 1.5rem;
  border-bottom-left-radius: 0;
}

/* Efeito hover */
.button {
  transition: border-radius 0.3s ease;
}

.button:hover {
  border-radius: 0.75rem;
}

Dica Prática

Use valores altos de border-radius (como 9999px) para criar cápsulas e círculos perfeitos. Combine com sombras (box-shadow) para efeitos de profundidade.

Inputs Personalizados

Os elementos de formulário podem ser estilizados para combinar com o design do seu site usando pseudo-classes como :focus, :hover e :checked.

Demonstração

Input de Texto

Input de Busca

Código Exemplo

/* Estilização básica de input */
input[type="text"],
input[type="email"],
input[type="password"] {
  width: 100%;
  padding: 0.75rem;
  border: 1px solid #d1d5db;
  border-radius: 0.5rem;
  transition: all 0.2s ease;
}

/* Estilo quando o input está em foco */
input:focus {
  outline: none;
  border-color: #6366f1;
  box-shadow: 0 0 0 3px rgba(99, 102, 241, 0.2);
}

/* Estilo para input de busca com ícone */
.search-input {
  padding-left: 2.5rem;
  background-image: url('search-icon.svg');
  background-position: 0.75rem center;
  background-repeat: no-repeat;
}

/* Checkbox personalizado */
.custom-checkbox {
  appearance: none;
  width: 1.25rem;
  height: 1.25rem;
  border: 2px solid #6366f1;
  border-radius: 0.25rem;
  cursor: pointer;
}

.custom-checkbox:checked {
  background-color: #6366f1;
  background-image: url("data:image/svg+xml,...");
}

Dica Prática

Mantenha a acessibilidade ao estilizar inputs: não remova completamente o outline no :focus, mas estilize-o de forma atraente. Use labels associadas a todos os inputs e forneça feedback visual para estados como inválido ou desabilitado.

React-Icons

Código Exemplo

// Instalação
npm install react-icons

// Uso básico
import { FaReact, FaHeart, FaStar } from 'react-icons/fa';
import { MdEmail } from 'react-icons/md';

function IconExample() {
  const [liked, setLiked] = useState(false);
  
  return (
    <div>
      <FaReact className="text-blue-500 text-4xl" />
      
      <button 
        onClick={() => setLiked(!liked)}
        className="flex items-center gap-2 px-4 py-2 rounded-lg"
      >
        {liked ? (
          <FaHeart className="text-red-500" />
        ) : (
          <FaHeart className="text-gray-400" />
        )}
        <span>{liked ? 'Curtido' : 'Curtir'}</span>
      </button>
      
      <div className="flex gap-4 mt-4">
        <FaStar className="text-yellow-400 text-2xl" />
        <MdEmail className="text-blue-400 text-2xl" />
      </div>
    </div>
  );
}
